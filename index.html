Things We Know about HTML

  • We need a "!DOCTYPE html" declaration at the top
  • We need an "html" tag after that
  • Following that, there is a "head" tag
  • Between the opening and closing head tags, there is a "title" tag

HTML—“HyperText Markup Language”—is the language used to tell your web browser what each part of a website is. So, using HTML, you can define headers, paragraphs, links, images, and more, so your browser knows how to structure the web page you're looking at.


(HTML) is a type of computer language that is used to create pages that can be posted on the Internet or sent via email.


HTML tables are used for displaying data that make sense in spreadsheet software. They consist of rows and columns and are often used on websites for the effective displaying of tabular data
